## General Guidelines

0. All ufabric-projects must follow the public available guidelines [this repository](https://raw.githubusercontent.com/ufabric-org/project-polaris/main/README.md).
1. All projects must exists in the [ufabric-org organization](https://github.com/ufabric-org) currently hosted in GitHub.
2. All projects are allowed to use AI for content, code, image generation according to the [AI - LLMs and other tools](ai-guidelines.md) guidelines.
3. All documents must be written in English according to the [language guidelines](language-guidelines.md).
4. All projects must follow the [technical guidelines](technical-guidelines.md).
5. All projects must have a CHANGELOG following the [keepachangelog.com/en/1.1.0](https://keepachangelog.com/en/1.1.0/) conventions.
